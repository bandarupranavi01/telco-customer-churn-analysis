Customer Churn Analysis Dashboard
📌 Project Overview

This project focuses on analyzing customer churn patterns using Power BI.

The dashboard provides insights into customer demographics, service subscriptions, contract types, billing behavior, and the key factors influencing customer churn.

The objective of this project is to help businesses identify customers likely to leave and understand the major reasons behind churn, enabling better customer retention strategies.

📊 Dashboard Highlights

1. Customer Demographic Analysis

2. Service Subscription Analysis
   
Churn is highest among Fiber Optic users

Customers with No Tech Support show higher churn

Majority customers use Phone Services

3. Contract & Billing Analysis
   
Month-to-Month contract customers have the highest churn

Electronic Check payment users show increased churn behavior

Long-term contract customers are more likely to stay

4. Churn Prediction & Key Drivers
   
Key factors affecting churn:

Contract Type

Internet Service Type

Tech Support Availability

Monthly Charges

Payment Method

📈 Key Insights

Customers with Month-to-Month contracts are more likely to churn.

Fiber Optic internet users have the highest churn rate.

Customers without Tech Support are at greater risk of leaving.

Higher monthly charges contribute significantly to customer churn.

Long-term contracts improve customer retention.
